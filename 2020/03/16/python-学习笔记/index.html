Python基础

Python中的编码规范

空行

  • 函数之间或类的方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分隔，以突出函数入口的开始。

  • 空行与代码缩进不同，空行并不是Python语法的一部分。书写时不插入空行，Python解释器运行也不会出错。但是空行的作用在于分隔两段不同功能或含义的代码，便于日后代码的维护或重构。

  • 记住：空行也是程序代码的一部分。

换行

Python通常是一行写一条语句，如果语句很长，可以使用\反斜杠来实现多行语句

python

在 []和{}或()中的多行语句不需要使用反斜杠

python

缩进

Python是使用缩进来表示代码块，不需要使用{}

缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的缩进空格数

如果缩进的空格数不一致，会导致运行错误

注释

单行注释：通过#开头

多行注释：通过'''三个单引号开始，以三个单引号结束'''

输出注释：通过函数的方法.__doc__来输出函数里的注释

代码组

缩进相同的一组语句构成一个代码块，我们称之代码组。

像if、while、def和class这样的复合语句，首行以关键字开始，以冒号( : )结束，该行之后的一行或多行代码构成代码组。

我们将首行及后面的代码组称为一个子句(clause)。

python

数据类型和变量

数据类型：

  • 整数

  • 浮点数（小数）

  • 字符串

    • 当括号内的许多字符需要转义的时候用\就不是一个很明智的选择
    • 可以选择使用r""，引号内的默认都不会转义
    • 特殊字符
      • \n：换行
      • \t：制表符
      • 如果字符串内部有很多换行，用\n写在一行里不好阅读，为了简化，Python允许用’’’…’’’的格式表示多行内容，可以自己试试：

  • 布尔值

    • not： 取反运算符，相当于！
    • and： 相当于与
    • or： 相当于或
    • None： 空值

  • 变量：

    1. python是动态语言
    2. 变量不用关键字来声明，直接a = 1，a就是整数类型

# 开头的是注释

2**10 === 2的10次方

字符编码

Python提供了几种方法来让

方法1：ord("A")：取A字符的整数表示

方法2：chr(66)：将66编码转换成字符

数组

List

List是有序的集合，可以随时添加和删除其中的元素

语法：classmates = ["Michael","Bob","Tracy"]

可以通过len函数来获取list元素的个数：len(classmates)

当索引超出了范围是，Python会报错IndexError，所以不要越界，极盗最后一个元素的索引是length -1

如果要获取最后一个元素，出了计算索引位置以外，还可以使用-1做索引，直接获取最后一个元素

classmates[-1]：获取数据的最后一个元素

数据的增删改查

classmates.append("Adam")：添加到数组的最后面

classmates.insert(index,"Jack")：添加到索引为index位置

classmates.pop()：删除数组最后一个元素

classmates[index] = "Sarah"：修改索引为index位置的值

classmates[index]：查看索引值为index位置的值

多维数组

语法：classmates = ["python","java",["asp","php"],"javascript"]

访问：classmates[2][0]

tuple

tuple一旦初始化就不能修改，定义的方式也不同

语法：classmates = ("Michael","Bob","Tracy")

他没有list的增删查改的方法，只能正常的去访问

能用tuple代替list就尽量使用，会保证数据更加安全

注意：

t = (1)

定义的不是tuple，是1这个数！这是因为括号()既可以表示tuple，又可以表示数学公式中的小括号，这就产生了歧义，因此，Python规定，这种情况下，按小括号进行计算，计算结果自然是1。
