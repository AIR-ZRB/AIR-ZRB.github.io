python学习笔记

python基础

数据类型和变量

  • 数据类型：

    1. 整数

    2. 浮点数（小数）

    3. 字符串

      • 当括号内的许多字符需要转义的时候用\就不是一个很明智的选择
      • 可以选择使用r""，引号内的默认都不会转义
      • 特殊字符
        • \n：换行
        • \t：制表符
        • 如果字符串内部有很多换行，用\n写在一行里不好阅读，为了简化，Python允许用’’’…’’’的格式表示多行内容，可以自己试试：

    4. 布尔值

      • not： 取反运算符，相当于！
      • and： 相当于与
      • or： 相当于或
      • None： 空值

  • 变量：

    1. python是动态语言
    2. 变量不用关键字来声明，直接a = 1，a就是整数类型
